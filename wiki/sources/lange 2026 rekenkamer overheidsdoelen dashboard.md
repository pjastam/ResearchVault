# lange 2026 rekenkamer overheidsdoelen dashboard

## Summary
De Algemene Rekenkamer stelt dat er een gebrek is aan een overheidsdashboard voor transparantie van overheidsbeleid en initieert zelf 'Blik op Nederland'. Dit dashboard toont de voortgang op diverse maatschappelijke doelen, waaronder zorgindicatoren, en benadrukt dat het kabinet zelf geen dashboard heeft, waardoor de overheid transparantie via externe instanties moet opvragen.
